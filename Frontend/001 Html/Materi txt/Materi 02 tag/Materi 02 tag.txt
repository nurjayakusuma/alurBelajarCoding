TAG HTML!

tag merupan bagian elemen pada bahasa pemrograman, tag terdiri dari elemen pembungkus dan didalamnya terdapat perintah untuk menjalankan perintah, contoh tag <html> content </html>

<title></title> = judul halaman
<style></style> = css
<script></script> = java script
<meta></meta> = meta data

contoh Tag pada head!

<head>

#Meta data, Peran meta data untuk mendeskripsikan website yang kita miliki.

	<meta charset="UTF-8">
	<meta name="description" content="kuliah pemrograman web">
	<meta name="author" content="Nurjaya kusuma">

</head>
